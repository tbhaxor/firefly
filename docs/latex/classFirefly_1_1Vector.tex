\doxysubsection{Detailed Description}
Represents a mathematical vector in n-\/dimensional space. 

The \doxylink{classFirefly_1_1Vector}{Vector} class provides various operations on vectors, such as addition, subtraction, scaling, dot product, cross product, normalization, angle calculation, and more.
